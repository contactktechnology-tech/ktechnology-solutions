# Documentation Index

This directory contains all formal documentation for KTechnology Solutions.

## Structure

- adr/           Architecture Decision Records
- architecture/  System and solution architecture
- policies/      Governance, compliance, and policy documents
- sops/          Standard Operating Procedures
- PROJECT_SCOPE.md  Authoritative scope definition

## Governance

- All documents in this directory are source-controlled
- Changes must be made via pull request
- `main` is protected and cannot be force-pushed
